⚡ 5-MINUTE QUICKSTART
═════════════════════════════════════════════════════════════════════════════

Step 1: Get API Key (2 min)
  → Visit: https://www.themoviedb.org/settings/api
  → Create account or login
  → Generate new API key

Step 2: Configure (1 min)
  → Open: movie-extension/utils/api.js
  → Find: API_KEY: 'YOUR_TMDB_API_KEY_HERE'
  → Replace with your actual key
  → Save!

Step 3: Load Extension (1 min)
  → Open: chrome://extensions/
  → Turn ON: "Developer Mode" (top right)
  → Click: "Load unpacked"
  → Select: movie-extension folder
  → Done!

Step 4: Test (1 min)
  → Go to: hotstar.com or primevideo.com
  → Click: Any movie detail page
  → Click: Extension icon
  → Click: "Show Recommendations"
  → See recommendations!

🆘 QUICK TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

  Q: "Extension won't load"
  A: Check if api.js has syntax errors. Reload at chrome://extensions

  Q: "No recommendations appearing"
  A: Verify API key is set in utils/api.js. Clear cache. Try again.

  Q: "Movie not detecting"
  A: Make sure you're on actual movie detail page, not search/home

  Q: "API error"
  A: Check internet connection. Verify API key is correct. Wait 1 min.

  More help → See START_HERE.md or README.md Troubleshooting section
