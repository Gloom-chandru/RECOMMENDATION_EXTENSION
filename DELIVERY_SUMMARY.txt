QUICK START (5 MINUTES)
================================================================================

1. Get API Key (2 min)
   → Visit: https://www.themoviedb.org/settings/api
   → Create account or login
   → Generate API key

2. Configure Extension (1 min)
   → Open: movie-extension/utils/api.js
   → Replace: API_KEY: 'YOUR_TMDB_API_KEY_HERE'
   → Save file

3. Load in Chrome (1 min)
   → chrome://extensions/
   → Developer Mode: ON
   → Load unpacked → select movie-extension/

4. Test (1 min)
   → Go to hotstar.com or primevideo.com
   → Click extension icon
   → Click "Show Recommendations"
